# skein

Indieweb static site generator

## Features

* Publish to Amazon S3, git, or local filesystem
* Supported post types:
  * Notes
    * Autolinked urls
    * OEmbedded youtube and soundcloud links
  * Replies
  * Likes
  * Reposts
  * Articles
    * Summary view in stream
  * Photos
  * Audio
* Tags
* Micropub
* Webmentions

## Running

```
node server.js config.json
```
